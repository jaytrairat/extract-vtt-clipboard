
# extract-vtt-clipboard
สำหรับ Copy ข้อมูลจาก Virustotal แล้วแปลง Format เพื่อให้วางบน Microsoft word ได้

## Installation
```bash
go install github.com/jaytrairat/extract-vtt-clipboard@latest
```

## Usage
```bash
extract-vtt-clipboard
```
